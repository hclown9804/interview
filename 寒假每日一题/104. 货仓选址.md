# 思路

* n是奇数选中位数；
* n是偶数选中间两个数都可以。

$$
|x-a|+|x-b| >= |a-b|
$$

# 时间复杂度

$$
nO(logn)
$$

# 代码

```c++
#include <iostream>
#include <algorithm>
using namespace std;

const int N = 100010;
int n, a[N];

int main() {
    cin >> n;
    for (int i = 0; i < n; ++i) cin >> a[i];
    sort(a, a + n);
    int ans = 0;
    for (int i = 0; i < n; ++i) ans += abs(a[i] - a[n /2 ]);
    cout << ans << endl;
    return 0;
}
```
