### [1833. 雪糕的最大数量](https://leetcode-cn.com/problems/maximum-ice-cream-bars/)

```java
class Solution {
    public int maxIceCream(int[] costs, int coins) {
        Arrays.sort(costs);
        int ans = 0;
        for (int i = 0; i < costs.length; ++i) {
            if (coins >= costs[i]) {
                ans++;
                coins -= costs[i];
            } else break;
        }
        return ans;
    }
}
```
